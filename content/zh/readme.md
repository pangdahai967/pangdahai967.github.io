# 胖大海的博客

你好，这里是我的个人博客。

我会在这里记录一些日常随笔、阅读笔记、学习过程和阶段性思考。内容不一定很多，但希望都是真实、清晰、值得回头再看的东西。

这里的很多想法都只是我在某个阶段的理解，之后也可能会继续修正、补充，甚至推翻。能留下这些变化，本身也是写作的意义之一。

主要内容大概会包括：

* 随笔与生活记录
* 阅读与观影感想
* 学习笔记与知识整理
* 一些临时冒出来的想法

这个博客的主要功能：

* 左侧可以看到我的基本信息和联系方式。
* 可以点击 RSS 图标订阅更新。
* 在 **Posts** 页面可以按时间查看文章。
* 可以通过分类、标签和搜索来查找内容。
* 右下角悬浮按钮可以调整阅读显示效果。

之后我也可能会继续补充更多内容，比如照片、收藏或专题整理。

*欢迎阅读。*
