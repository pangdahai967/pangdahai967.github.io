# 宇宝饭

你好，这里是我的博客。不定期发布一些内容，主要分为随笔、观点、知识三类。将来如果画了画也会放在这里。

需注意，这里所有观点都是阶段性的，很可能过段时间就被我推翻了。新的想法也会标注进关联博文里以供参考。


主要功能说明：

* 主页左侧可以联络作者，或点击 RSS 图标订阅最新文章。
* 从 Overview 切换至 Posts 栏可以按时间线查看文章。
* Posts 栏下可按分类或标签筛选文章。
* 每页左上方可用二字及以上字数的关键词检索文章。
* 右下方悬浮按钮可调出棕色滤镜、调节字间距和行间距、开启阅读进度条、切换字体。
* 每篇文章底下带有一个评论区，但用不了。如需讨论可以前往[Issues](https://github.com/ying-ya-rebirth/ying-ya-rebirth.github.io/issues)，点**new issue**开始发言即可。
  * 好心人也可以告诉我gitalk评论区哪里设置得有问题（我求你了🥲）
  * 给我发邮件也是完全OK的。
* 主页下方可以查看活跃日期和最近动向。
